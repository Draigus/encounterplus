# Group

```xml
<group id="987C39B8-2A3B-4B11-8503-6B3B1AFBBF02" sort="3">
    <name>Treasure</name>
    <slug>Treasure</slug>
</group>
```

## Attributes

| Attribute | Type | Description |
| --------- | ---- | ----------- |
| id  | String  | UUIDv4 |
| parent  | String  | UUID of parent item |
| sort  | String  | for sorting |

## Fields

| Field  | Type | Description |
| ------ | ---- | ----------- |
| name  | String | |
| slug  | String | for referencing |
